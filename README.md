## Getting started
### Install Qt
1. Download the Qt Online installer
2. Run the Qt Maintenance tool (installer) and install the following components:
* Qt
  * Qt 6.2.x
    * Desktop gcc 64-bit
    * Qt Shader Tools
    * Additional Libraries
      * Qt 3D
  * Developer and Designer Tools
    * Qt Creator
    * CMake
    * Conan
    * Ninja

### Open project
1. Start Qt Creator
2. Open CMakeLists.txt
